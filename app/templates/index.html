How it works:
  • Paste HTML or upload an HTML file
  • Select how long the link should be valid
  • Get a shareable link to your page
  • Expired pages are automatically deleted
Upload via curl:

# Upload a file (7 days expiration)
curl -X POST -F "html_file=@mypage.html" -F "expiration=7" {{ base_url }}/api/upload

# Upload from stdin (1 day expiration)
echo "<h1>Hello</h1>" | curl -X POST -F "html_file=@-;filename=page.html" -F "expiration=1" {{ base_url }}/api/upload

# Indefinite (expiration=0)
curl -X POST -F "html_file=@mypage.html" -F "expiration=0" {{ base_url }}/api/upload
